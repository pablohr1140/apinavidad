# Índice de documentación

## Contenido
- `architecture.md`: Arquitectura de alto nivel, capas, patrones y testing.
- `prisma-migration-plan.md`: Estrategia de migración de TypeORM a Prisma.
- `prisma-rollout.md`: Checklist y operación para habilitar Prisma con flags.
- `prisma-integration-tests.md`: Guía de pruebas integradas/e2e con Prisma (SQL Server incluido).
- `prisma-writes-plan.md`: Plan para habilitar escrituras Prisma y su validación.

## Uso
- Lee `architecture.md` para entender el modelo por capas y convenciones.
- Consulta `prisma-migration-plan.md` para el roadmap de migración.
- Sigue `prisma-rollout.md` y `prisma-writes-plan.md` al activar flags en entornos.
- Usa `prisma-integration-tests.md` para correr las suites Prisma y validar contra SQL Server.
